Deskripsi Proyek

Garbage Image Classification adalah aplikasi berbasis Artificial Intelligence (AI) yang mampu mengklasifikasikan jenis sampah menjadi tiga kategori utama: Plastik, Gelas, dan Metal.

Pengguna cukup mengunggah foto sampah melalui browser, lalu sistem akan secara otomatis mengklasifikasikan jenis sampah berdasarkan model yang sudah dilatih. Proyek ini bertujuan untuk menunjukkan bagaimana teknologi AI dapat diintegrasikan ke dalam kehidupan sehari-hari untuk mendukung pengelolaan sampah yang lebih cerdas dan berkelanjutan.

Model dilatih menggunakan FastAI dan kemudian di-deploy ke web menggunakan TensorFlow.js, sehingga aplikasi ini dapat dijalankan langsung di browser tanpa memerlukan server tambahan atau instalasi khusus.

Dataset

Proyek ini menggunakan dataset publik dari Kaggle: Garbage Classification V2

Dataset ini berisi ribuan gambar sampah yang dikelompokkan ke dalam 3 kelas:

  • Plastik
  • Gelas
  • Metal

Dataset ini menjadi dasar pelatihan model sehingga mampu mengenali perbedaan visual antar kategori dengan tingkat akurasi yang baik.

Project Description

Garbage Image Classification is an AI-powered application capable of classifying waste into three main categories: Plastic, Glass, and Metal.

Users can simply upload a photo of waste through the browser, and the system will automatically classify it based on the trained model. This project aims to demonstrate how AI can be integrated into everyday life to support smarter and more sustainable waste management.

The model was trained using FastAI and deployed to the web with TensorFlow.js, allowing the application to run directly in the browser without requiring any additional servers or installations.

Dataset

This project uses a public dataset from Kaggle: Garbage Classification V2

The dataset contains thousands of waste images categorized into three classes:

  • Plastic
  • Glass
  • Metal

This dataset forms the foundation of the training process, enabling the model to distinguish visual differences between categories with a solid level of accuracy.
